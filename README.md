# Bulolayer v2.0
Layer swap addon for TBC Classic Anniversary Edition

AUTO INVITE FOR LAYER SWAP

## Commands
/l or /layer - Find layer swap group
/layerconfig - Open settings panel
/layerclear - Clear all cooldowns
/layerstats - Show swap statistics
/layerfav [name] - Add/remove player from favorites
/layerhelp - Show all commands

## Features
- Auto-invite and auto-accept for layer swapping
- Minimap button with status indicator
- Configurable cooldown between swaps
- Persistent cooldowns (survive reload)
- Statistics tracking (swaps per day/total)
- Favorites system for trusted players
- Auto-leave option after swap
- Guild/Friends only mode
- Rate limiting to prevent spam
- Works with custom channels

## Settings
- Enable/Disable addon
- Custom invite message and channel
- Cooldown duration
- Party size threshold
- Guild chat / Whisper support
- Auto-leave with delay
- Verbose notifications

Left-click minimap button to open settings
Right-click to toggle addon on/off
Drag to reposition
